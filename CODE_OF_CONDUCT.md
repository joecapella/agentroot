# Contributor Covenant Code of Conduct

We follow the Contributor Covenant. Summary:

- Be friendly and welcoming.
- Be respectful and considerate in all communications.
- Embrace constructive criticism and assume good faith.

If you experience or witness unacceptable behavior, contact the maintainers
so we can address it.

For the full text, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
