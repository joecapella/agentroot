# Contributing to AgentRoot

Thanks for your interest in contributing! We welcome bug reports, feature
requests, and pull requests.

- Fork the repository and create a feature branch.
- Run tests and linters locally before submitting a PR: `npm test`.
- Keep changes focused and add tests for new behavior.
- Use conventional commit messages (e.g., `fix:`, `feat:`, `chore:`).
- All PRs should include a short description of the change and any manual
  testing steps.

For major changes, open an issue first to discuss the design.
